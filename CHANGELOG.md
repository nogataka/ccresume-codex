# Changelog

## [Unreleased] - 2025-10-14
### Added
- プロジェクト一覧とセッション一覧にインタラクティブ検索モードを実装し、`/` でトグル、`Ctrl+U` でクリアできるようにしました。
- 検索時にプロジェクト名・パスやセッションのメッセージ本文を対象にフィルタリングできるよう、会話データのキャッシュ取得を追加しました。
- Ink コンポーネント（ProjectList / ConversationList）のヘッダに検索結果件数やクエリ表示を追加し、0 件時のメッセージを改善しました。
- プロジェクト／セッション検索 UI の回 regress 防止のためのテスト (`ProjectList.test.tsx`, `ConversationList.test.tsx`) を追加しました。

### Changed
- セッションリストのナビゲーションロジックを検索モードとページングで切り替えるよう再構成し、検索中はページング操作を無効化しました。
- 新しい検索キーバインドを `defaultConfig` に追加し、既存のキーハンドリングを拡張しました。
- 設定ファイルの既定パスを `~/.config/ccresume-codex/config.toml` に更新し、サンプル設定 (`config.toml.example`) を Codex 用に刷新しました。

### Fixed
- 検索モード終了時に選択状態やステータスがリセットされない問題を防ぐため、終了時にカーソルとステータスメッセージを再初期化するようにしました。
